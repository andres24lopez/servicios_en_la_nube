# Sistema de Construcción con Node.js y MySQL

## Requisitos
- Node.js
- MySQL
- Visual Studio Code

## Instalación

### 1. Crear base de datos
Ejecuta el archivo:

database.sql

en MySQL.

### 2. Instalar dependencias
Abre la terminal en la carpeta del proyecto y ejecuta:

npm install

### 3. Configurar MySQL
Edita el archivo:

server.js

y cambia:

host
user
password

según tu MySQL.

### 4. Ejecutar proyecto

npm start

### 5. Abrir en navegador

http://localhost:3000

## Railway
Para Railway:
- Sube el proyecto a GitHub
- Conecta el repositorio en Railway
- Agrega las variables de entorno de MySQL

PASOS PARA PUBLICAR EN RAILWAY

1. DESCOMPRIME EL ZIP

2. COPIA TODO A TU REPOSITORIO GITHUB

3. HAZ PUSH A GITHUB

4. EN RAILWAY:
   - NEW PROJECT
   - DEPLOY FROM GITHUB REPO
   - SELECCIONA TU REPOSITORIO

5. AGREGAR BASE DE DATOS:
   - ADD SERVICE
   - DATABASE
   - POSTGRESQL

6. EN POSTGRESQL:
   - OPEN
   - QUERY

7. EJECUTA database.sql

8. RAILWAY DETECTARÁ AUTOMÁTICAMENTE:
   PGHOST
   PGPORT
   PGDATABASE
   PGUSER
   PGPASSWORD

9. GENERAR DOMINIO:
   SETTINGS
   NETWORKING
   GENERATE DOMAIN

10. LISTO

TU APP ESTARÁ ONLINE.
